📋 LendingPlate Requirements
Required CSV Columns:
  • mobile, phoneNumber, phonenumber, or phone_number - 10 digit mobile without country code
  • pincode, pinCode, or pin_code - Exactly 6 digits
  • net_mothlyincome, net_monthly_income, monthly_income, monthly income, or income - Numeric value between 1 and 1,000,000
  • pan, pan_number, or pancard - PAN card number (will be converted to uppercase)
  • dob or date_of_birth - Format: YYYY-MM-DD or DD/MM/YYYY
  • first_name + last_name OR name - Customer name (max 50 chars)
  • profession (optional) - Salaried, Self-Employed, or leave blank (defaults to Salaried)
  • ref_id, reference_id, or id (optional) - 5-10 digit unique reference ID, auto-generated if absent
Validation Rules:
  • Mobile: Must be exactly 10 digits (non-digits are stripped)
  • Pincode: Must be exactly 6 digits (non-digits are stripped)
  • Monthly Income: Must be numeric without commas (commas are removed)
  • DOB: Auto-converts YYYY-MM-DD → DD/MM/YYYY format
Configuration Required:
  • Environment Variables:
    • LENDINGPLATE_PARTNER_ID - Your partner ID (e.g., TESTING, 3115, CREDNORTH)
    • LENDINGPLATE_API_TOKEN - API authentication token (raw token, no Bearer prefix)
    • LENDINGPLATE_ENV - production or uat (defaults to production)
API Endpoint: https://lms.lendingplate.co.in/api/v2/partner/generate_lead
